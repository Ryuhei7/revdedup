\subsection{\-Detailed \-Description}
\hyperlink{structIndirect}{\-Indirect} entry in the image recipe. \-It only exists when the image got reversely deduplicated. \-It works by referencing the segment, the chunk offset and the number of chunks of the future version (if restoring version i, then entries are pointing to version i + 1.

\-In restore, the use of len will be replaced to store the version number of future image that contains the referenced segment.
