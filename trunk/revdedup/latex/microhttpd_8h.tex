\subsection{\-Detailed \-Description}
public interface to libmicrohttpd \begin{DoxyAuthor}{\-Author}
\-Christian \-Grothoff 

\-Chris \-Gauthier\-Dickey
\end{DoxyAuthor}
\-All symbols defined in this header start with \-M\-H\-D. \-M\-H\-D is a small \-H\-T\-T\-P daemon library. \-As such, it does not have any \-A\-P\-I for logging errors (you can only enable or disable logging to stderr). \-Also, it may not support all of the \-H\-T\-T\-P features directly, where applicable, portions of \-H\-T\-T\-P may have to be handled by clients of the library.

 continue\char`\"{} are understood and handled automatically).
